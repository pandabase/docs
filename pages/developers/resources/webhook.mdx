# Webhook

You can configure webhook endpoints via the API to be notified about events that
happen in your Pandabase shop.

### Endpoints

| METHOD |              URL              |         DESCRIPTION         |             CHANGELOG SHORT CODE |
| :----- | :---------------------------: | :-------------------------: | -------------------------------: |
| GET    |   `/shop/:shop_id/webhooks`   | List all webhook endpoints  |         `GET#/SHOP/{ID}/WEBHOOK` |
| GET    | `/shop/:shop_id/webhooks/:id` | Retrieve a webhook endpoint |    `GET#/SHOP/{ID}/WEBHOOK/{ID}` |
| POST   |   `/shop/:shop_id/webhooks`   |  Create a webhook endpoint  |        `POST#/SHOP/{ID}/WEBHOOK` |
| DELETE | `/shop/:shop_id/webhooks/:id` |  Delete a webhook endpoint  | `DELETE#/SHOP/{ID}/WEBHOOK/{ID}` |

#### List all webhook endpoints

`/shop/:shop_id/webhooks`

Returns a list of your webhooks.
