# Webhook

You can configure webhook endpoints via the API to be notified about events that
happen in your Pandabase shop.

### Endpoints

| METHOD |              URL               |                 DESCRIPTION |
| :----- | :----------------------------: | --------------------------: |
| GET    |   `/shops/:shop_id/webhooks`   |  List all webhook endpoints |
| GET    | `/shops/:shop_id/webhooks/:id` | Retrieve a webhook endpoint |
| POST   |   `/shops/:shop_id/webhooks`   |   Create a webhook endpoint |
| DELETE | `/shops/:shop_id/webhooks/:id` |   Delete a webhook endpoint |

#### List all webhook

`/shop/:shop_id/webhooks`

Returns a list of your webhooks.
