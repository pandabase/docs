# Order

| METHOD |                     URL                     |                DESCRIPTION |
| :----- | :-----------------------------------------: | -------------------------: |
| GET    |    `/shops/:id/public/orders/:order_id`     |    Retrive an public order |
| GET    |          `/shops/:shop_id/orders`           |            List all orders |
| GET    |        `/shops/:shop_id/orders/:id`         |          Retrieve an order |
| GET    |    `/shops/:shop_id/orders/intents/:id`     |   Retrieve an order intent |
| POST   | `/shops/:shop_id/orders/intents/initialize` | Initialize an order intent |
| POST   | `/shops/:shop_id/orders/intents/create/:id` |     Create an order intent |

#### Retrive an public order

`/shop/:id/public/orders/:order_id`

#### List all orders

`/shop/:shop_id/orders`

#### Retrieve an order

`/shop/:shop_id/orders/:id`

#### Retrieve an order intent

`/shop/:shop_id/orders/intents/:id`

#### Initialize an order intent

`/shop/:shop_id/orders/intents/initialize`

#### Create an order intent

`/shop/:shop_id/orders/intents/create/:id`
