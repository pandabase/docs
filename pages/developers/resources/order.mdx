# Order

| METHOD |                    URL                     |        DESCRIPTION         |                         CHANGELOG SHORT CODE |
| :----- | :----------------------------------------: | :------------------------: | -------------------------------------------: |
| GET    |    `/shop/:id/public/orders/:order_id`     |  Retrive an public order   |          `GET#/SHOP/{ID}/PUBLIC/ORDERS/{ID}` |
| GET    |          `/shop/:shop_id/orders`           |      List all orders       |                 `GET#/SHOP/{ID}/ORDERS/{ID}` |
| GET    |        `/shop/:shop_id/orders/:id`         |     Retrieve an order      |                 `GET#/SHOP/{ID}/ORDERS/{ID}` |
| GET    |    `/shop/:shop_id/orders/intents/:id`     |  Retrieve an order intent  |         `GET#/SHOP/{ID}/ORDERS/INTENTS/{ID}` |
| POST   | `/shop/:shop_id/orders/intents/initialize` | Initialize an order intent |  `POST#/SHOP/{ID}/ORDERS/INTENTS/INITIALIZE` |
| POST   | `/shop/:shop_id/orders/intents/create/:id` |   Create an order intent   | `POST#/SHOP/{ID}/ORDERS/INTENTS/CREATE/{ID}` |

#### Retrive an public order

`/shop/:id/public/orders/:order_id`

#### List all orders

`/shop/:shop_id/orders`

#### Retrieve an order

`/shop/:shop_id/orders/:id`

#### Retrieve an order intent

`/shop/:shop_id/orders/intents/:id`

#### Initialize an order intent

`/shop/:shop_id/orders/intents/initialize`

#### Create an order intent

/shop/:shop_id/orders/intents/create/:id
